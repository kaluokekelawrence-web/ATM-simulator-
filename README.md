# ATM Simulator

A Python CLI app simulating core ATM operations.

## Features
- PIN authentication with 3 attempt limit
- Balance tracking
- Deposit and withdrawal
- Input validation
- Menu navigation

## Tech Stack 
- Python 3

## How to run
```bash
git clone https://github.com/kaluokekelawrence-web/ATM-simulator-
python main.py
```
